# ツールモデル

ツールモデルは、AIアシスタントが使用可能な操作ツールを表現します。ファイル操作、コマンド実行、検索などの機能をAIが利用するための定義を提供します。

## 属性

### 名前

ツールの名前を示します。1〜50文字の範囲内である必要があります。

### 説明

ツールの機能や使用方法を説明します。AIがツールの目的を理解するために使用されます。

### パラメータ

ツールが受け取ることができるパラメータの集まりです。各パラメータはツールの実行に必要な入力情報を定義します。

### 承認必須

ツールの実行にユーザーの承認が必要かどうかを示します。セキュリティ上重要な操作を行うツールには通常承認が必要と設定されます。

### 処理関数

ツールのロジックを実装します。パラメータを受け取り、ツールの実際の処理を実行します。

## パラメータ型

### 名前

パラメータの名前を示します。1〜50文字の範囲内である必要があります。

### タイプ

パラメータのデータ型を示します。'文字列'、'数値'、'真偽値'、'オブジェクト'、'配列'のいずれかの値を取ります。

### 説明

パラメータの目的や想定される値を説明します。AIがパラメータの適切な使用方法を理解するために使用されます。

### 必須

パラメータが必須かどうかを示します。必須の場合、ツール実行時に値の指定が必要になります。

### デフォルト値

パラメータが指定されなかった場合に使用されるデフォルト値です。任意項目であり、必須でない場合に意味を持ちます。

## ビジネスルール

- ツール名は一意である必要がある
- 必須パラメータは実行時に必ず指定する必要がある
- パラメータは指定された型に適合する必要がある
- 承認が必要なツールは、ユーザーの承認なしに実行できない
- ツールの実行結果は適切にフォーマットされる必要がある
